# 外觀模式(Facade)

## 題目

設計一個咖啡店管理系統，該系統包含以下子系統：

1. **CoffeeMachine**：咖啡機，可以製作不同種類的咖啡。
2. **MilkFrother**：牛奶發泡機，可以發泡牛奶。
3. **SyrupDispenser**：糖漿分配器，可以添加不同口味的糖漿。
4. **Lights**：燈光控制，可以調節燈光亮度。

使用外觀模式（Facade Pattern）來簡化咖啡店操作，使得店員可以通過簡單的接口來製作各種咖啡飲品。

請設計並實現以下功能：
- 拿鐵（Latte）：需要咖啡和牛奶泡沫。
- 卡布奇諾（Cappuccino）：需要更多的牛奶泡沫和咖啡。
- 焦糖瑪奇朵（Caramel Macchiato）：需要咖啡、牛奶泡沫和焦糖糖漿。
